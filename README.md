# Dizgin Rent A Car Landing Page

This repository contains a conversion-focused landing page for **Dizgin Rent A Car & Turizm**.

## Highlights

- English/Turkish language toggle (EN/TR)
- Funnel-style booking journey with clear CTAs
- Lead capture form that opens a pre-filled WhatsApp request

## Files

- `index.html` — main landing page
- `styles.css` — page styling
- `script.js` — language switcher, WhatsApp form funnel, and dynamic year

## Run locally

Open `index.html` directly in your browser, or use a simple local server:

```bash
python3 -m http.server 8080
```

Then visit:

`http://localhost:8080`
